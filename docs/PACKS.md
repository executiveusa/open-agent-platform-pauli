# Template Packs

Template packs live in `/packs/<pack_id>` and are versioned bundles with deterministic defaults, workflows, policies, and copy.

## Required files
- pack.json
- onboarding.json
- defaults.json
- workflows.json
- policies.json
- copy.json

Pack schema validation is enforced by `services/api/app/packs/pack_schema.json`.
